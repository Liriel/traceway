01

Principles

The four rules that keep Traceway's UI from drifting into generic-SaaS territory. When in doubt, pull the interface toward these.

// 01

Mono over prose.

Monospace everywhere. Labels read like code comments. Eyebrows use // prefix.

// 02

Deep ink, loud signal.

Surfaces are near-black. Color is reserved — when you see it, it means something is live, ok, warning, or critical.

// 03

Ship the data, not the chrome.

Thin hairlines, minimal shadows, no gradient bloat. The telemetry should out-brighten the frame around it.

// 04

Every pixel tokenized.

Colors, spacing, type, radii — all CSS variables. Swap the palette via --a1..--a4 and nothing else moves.

Motion

Traceway motion is brief and physical. Hover states settle in 120–150ms. Panels and menus use a snappier 200ms entrance curve; exits are linear and 150ms.

Standard
cubic-bezier(.2,.8,.2,1) · 200ms

Buttons, chips, hovers, hairline shifts.

Entrance
cubic-bezier(0,0,.2,1) · 150ms

Mega menus, tooltips, modals.

Exit
cubic-bezier(.4,0,1,1) · 120ms

Dismissal — snap back to rest.

07

Animation

On top of the base motion tokens, the landing page wears a GSAP + ScrollTrigger polish layer. Each recipe is scoped, brief, and gated on prefers-reduced-motion. Goal: the page feels like glass settling — never like a demo reel.

// The philosophy

Restraint over flash.

Animations exist to guide attention and reward scroll — not to prove the site is "alive." One motion per element. Nothing longer than 1100ms. Everything gated on a reduced-motion query.

One entrance. Each element animates in once, never again. No re-triggering on scroll-up.
Layered, not simultaneous. Hero elements cascade over ~1.1s with 60–150ms stagger. The brain reads order.
Scroll rewards, never gates. If JS fails, every section is still readable. [data-reveal] is progressive.
Physics, not transitions. Magnetic buttons use elastic recoil. Tilt cards use 3D perspective. The feel of mass.

Scroll reveal primary

Every section heading + grid enters once on scroll. Y-offset 24–36px, opacity 0→1, 0.85s, power3.out. Never re-triggers.

// ScrollTrigger — fire once, never reverse
gsap.set(els, { opacity: 0, y: 28 });
ScrollTrigger.create({
  trigger: sec,
  start: 'top 78%',
  once: true,
  onEnter: () => gsap.to(els, {
    opacity: 1, y: 0, duration: 0.85,
    ease: 'power3.out', stagger: 0.08,
  }),
});
Stagger cascade primary

Grids (bento, features, deploy) reveal child-by-child. stagger: 0.07s gives the eye time to land before the next arrives. Never uniform, never instant.

gsap.to(cards, {
  opacity: 1, y: 0, duration: 0.9,
  ease: 'power3.out',
  stagger: { each: 0.07, from: 'start' },
});
Scrub parallax primary

The hero dashboard drifts −60px as you scroll past. scrub: 0.8 smooths scroll-tied motion so it never feels mechanical.

gsap.to('#dashboard-mount', {
  y: -60, ease: 'none',

    start: 'top top',
    end: 'bottom top',

Magnetic CTA physics

Primary & accent buttons track the cursor within a 18px radius, then snap back with elastic.out(1, 0.5). Signals the site rewards interaction.

btn.addEventListener('mousemove', e => {
  const r = btn.getBoundingClientRect();
  const x = (e.clientX - r.left - r.width/2) / (r.width/2);
  const y = (e.clientY - r.top - r.height/2) / (r.height/2);
  gsap.to(btn, { x: x*18, y: y*18,
    duration: 0.4, ease: 'power3.out' });
});

// Do & don't
✓ Do
  • Use once: true on content reveals — re-triggering on scroll-up is distracting.
  • Set initial hidden state with gsap.set() before render to prevent FOUC.
  • Default to power3.out for entrances. It's the house curve.
  • Gate every animation on prefers-reduced-motion — early return the whole script.
  • Call ScrollTrigger.refresh() after React/async content mounts.
✗ Don't
  • Animate the same element twice. One entrance, one hover — nothing else.
  • Chain durations over 1.2s. The eye will move on.
  • Use bounce or back eases on text. Reserved for physical objects (buttons, cards).
  • Fade in everything. If it doesn't earn an animation, let it be static.
  • Animate a scroll-linked element without scrub. Janky framerate will show.
